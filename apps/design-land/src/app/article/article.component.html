Article

Daffodil's article component provides styles to common element selectors to create an article in a content page.

The daff-article component can be used on any content pages that display long-form text-driven information. It is meant to be used as a standalone element. It should not be nested inside any other components that may change the background color from the anticipated one. In the event that you must nest a daff-article inside another component, please ensure that you set the article's background color to the default body color.

Headings

The article component provides designs for the h1 through h6 headings.

Article Lead

Lead is used for the opening paragraph of an article. It should be used to provide a summary or leading information for an article.

Article Meta

The article meta directive can be used to highlight metadata about your article (i.e. author name, date, etc).

Link

The link style in an article uses the default browser link style.

Code

Since articles are often prone to containing code blocks, we also provide support for inline and multiline blocks of code.

Inline code

Use inline code elements where you would like to highlight a short snippet of code, like an individual function name or a type.

Code blocks

Use code blocks within the article to render a chunk of code that can be easily consumed by a fellow developer.

<pre>s are used for multiple lines of code. A scrollbar will appear if you choose to set a max-height or max-width to your code block.

Horizontal Rules

Horizontal Rules can be use to separate sections of content.

Blockquote

Blockquotes are used to quote blocks of content.
